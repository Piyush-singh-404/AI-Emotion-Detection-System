# AI Emotion Detection System

This project is a real-time facial emotion detection system developed using Python and Artificial Intelligence techniques. The system captures live video through a webcam, detects human faces, and predicts emotional expressions such as happy, sad, angry, neutral, fear, and surprise.

## Project Overview

The main objective of this project is to demonstrate how artificial intelligence can be applied in human emotion recognition using facial expressions. The system processes live camera input and identifies emotions instantly.

## Key Features

* Real-time webcam-based face detection
* Emotion prediction using facial analysis
* Automatic screenshot capture
* Emotion data storage in Excel sheet
* Graph generation for emotion analysis

## Technologies Used

* Python
* OpenCV
* FER Library
* OpenPyXL
* Matplotlib

## Project Files

* emotion.py
* graph.py
* emotion_log.xlsx
* graph.png
* screenshots folder

## Working Process

1. Webcam captures live face image
2. Face is detected frame by frame
3. Emotion is predicted using AI model
4. Data is stored in Excel file
5. Graphs are generated for analysis

## Output Emotions Detected

* Happy
* Sad
* Angry
* Neutral
* Fear
* Surprise

## Future Scope

The project can be extended with voice analysis, attendance integration, dashboard monitoring, and advanced AI analytics.
